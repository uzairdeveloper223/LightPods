Capabilities

Everything your clone pods need

Built by reverse-engineering Apple's proprietary BLE Continuity Protocol and HFP vendor extensions.

battery_full

Per-Pod Battery

Real battery levels for left pod, right pod, and case — decoded from Apple's BLE proximity pairing nibbles, not generic HFP guesses.

mic

Microphone Detection

Identifies which pod currently holds the active microphone by parsing the status byte's primary pod bit and case flip logic.

hearing

Ear Detection

Knows when each pod is in your ear or out, using the BLE status bits — the same data Apple uses for automatic play/pause.

notifications_active

System Overlay

iOS-style connection popup appears when your pods are detected nearby. Smart dismiss logic — won't spam you after you close it.

speed

Background Monitoring

Lightweight foreground service with battery optimization exemption. Live battery in the notification bar, even when the app isn't open.

palette

Material You

Dynamic color theming on Android 12+, with manual dark/light/system toggle. Every pixel follows Material Design 3 specs.
